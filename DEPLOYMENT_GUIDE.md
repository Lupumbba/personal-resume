# 个人简历网站部署指南

## 1. 注册Vercel账号

1. 访问 [Vercel官网](https://vercel.com/)
2. 点击右上角的 "Sign Up" 按钮
3. 选择使用GitHub账号登录（推荐，方便后续连接仓库）
4. 按照提示完成注册流程

## 2. 连接GitHub仓库

1. 登录Vercel后，点击右上角的 "Add New" 按钮，选择 "Project"
2. 在 "Import Git Repository" 页面，点击 "GitHub" 图标
3. 授权Vercel访问你的GitHub账号
4. 在仓库列表中找到并选择 `Lupumbba/personal-resume` 仓库
5. 点击 "Import" 按钮

## 3. 配置部署设置

1. 在 "Configure Project" 页面，保持默认配置即可：
   - Framework Preset: `Other`
   - Root Directory: 保持为空（使用项目根目录）
   - Build Command: 保持为空（静态网站不需要构建）
   - Output Directory: 保持为空（静态网站直接使用根目录）
2. 点击 "Deploy" 按钮开始部署

## 4. 完成部署

1. 等待部署完成（通常需要1-2分钟）
2. 部署成功后，Vercel会显示你的网站URL（例如：`https://personal-resume.vercel.app`）
3. 点击该URL即可访问你的个人简历网站

## 5. 配置自定义域名（可选）

如果你有自己的域名，可以按照以下步骤配置：

1. 在Vercel项目页面，点击 "Settings" 选项卡
2. 选择 "Domains" 菜单
3. 输入你的域名（例如：`resume.example.com`）
4. 按照Vercel的提示在你的域名注册商处配置DNS记录
5. 等待DNS记录生效后，你的网站即可通过自定义域名访问

## 6. 后续更新

当你需要更新网站内容时，只需：
1. 在本地修改代码
2. 提交并推送到GitHub仓库
3. Vercel会自动检测到代码变更并重新部署网站

## 7. 测试网站

部署完成后，请测试网站的以下功能：
1. 检查所有页面内容是否正确显示
2. 测试导航菜单是否正常工作
3. 检查响应式设计在不同设备上的表现
4. 测试动画效果是否正常播放
5. 检查网站加载速度

## 8. 常见问题

### 8.1 部署失败
- 检查GitHub仓库是否有正确的文件结构
- 确保index.html文件存在于仓库根目录
- 查看Vercel部署日志获取详细错误信息

### 8.2 网站访问缓慢
- 优化图片大小
- 确保CSS和JavaScript文件已压缩
- 考虑使用CDN加速

### 8.3 自定义域名无法访问
- 检查DNS记录是否正确配置
- 等待DNS记录生效（通常需要10-30分钟）
- 确保域名已在Vercel中正确添加
